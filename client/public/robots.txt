# POLICY
# ──────
# Every public, content-heavy page is open to crawlers. Authenticated paths
# (login, dashboard, admin, submission, subscriber/attorney portals) and the
# JSON/tRPC API are closed. This applies equally to:
#
#   • Search engines         (Google, Bing, DuckDuckGo, Yandex, Baidu, Apple)
#   • AI training / answering bots
#                            (GPTBot, ChatGPT-User, OAI-SearchBot, ClaudeBot,
#                             Claude-Web, anthropic-ai, Google-Extended,
#                             Applebot-Extended, PerplexityBot, Perplexity-User,
#                             CCBot, YouBot, Amazonbot, Diffbot)
#   • Social preview bots    (Twitterbot, facebookexternalhit, LinkedInBot,
#                             Slackbot, Discordbot, TelegramBot, WhatsApp)
#   • Archive & monitoring   (ia_archiver, Wayback, UptimeRobot, Pingdom,
#                             Site24x7)
#
# The `User-agent: *` line at the end of the list catches every bot that
# doesn't have its own named stanza. RFC 9309 lets one rule-block apply to
# multiple User-agent lines, so this is both explicit and DRY.
#
# If you add a new authenticated path to the client router, add it to the
# Disallow list below.
