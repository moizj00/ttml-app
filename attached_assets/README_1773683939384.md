# Animation Precision Master for Replit

This pack gives you a reusable agentic skill focused on **best-in-class motion for a lawyer website**.

## What it is
A motion-direction skill that tells Replit Agent how to:
- audit UI before animating it
- apply premium, trust-building animations
- stay accessible and performance-safe
- avoid flashy or low-trust effects
- work especially well for legal landing pages, dashboards, auth, pricing, and review workflows

## Files
- `SKILL.md` — the core skill instructions
- `replit-paste-prompt.md` — quick prompt you can paste into Replit Agent
- `animation-checklist.md` — review checklist for each page or component

## Best use cases
- homepage hero improvements
- practice area cards
- pricing section motion
- sticky header behavior
- auth/signup/login polish
- letter-generation workflow UI
- attorney review center microinteractions
- modal/preview/paywall transitions

## Tone of motion
Think:
- premium
- credible
- precise
- calm
- expensive

Not:
- flashy
- bouncy
- hyperactive
- “look mom, I discovered keyframes”

## How to use in Replit
Open Replit Agent and paste the content from `SKILL.md` into your custom skill/instruction area, or use the quick prompt from `replit-paste-prompt.md`.

Then give tasks like:
- “Apply Animation Precision Master to the homepage hero and navbar.”
- “Use this skill to polish the attorney dashboard with restrained motion.”
- “Improve auth form transitions using this skill and keep reduced-motion support.”
- “Refine the letter preview modal using premium legal-brand motion.”

## Recommended request pattern
Ask Replit Agent to return:
1. motion audit
2. animation plan
3. exact files to edit
4. code changes
5. reduced-motion fallback
6. performance notes
