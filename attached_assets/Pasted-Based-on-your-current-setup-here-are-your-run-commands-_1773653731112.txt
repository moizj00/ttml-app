Development Server
pnpm run dev
# or npm run dev
What it does: Runs NODE_ENV=development tsx watch server/_core/index.ts — watches for file changes and hot-reloads the server. Vite dev server starts automatically for the frontend on port 5000.

Production Server
First, build the application:

pnpm run build
# or npm run build
What it does: Compiles the frontend (Vite) + bundles the backend (esbuild), outputs to dist/ folder.

Then start the production server:

pnpm run start
# or npm run start
What it does: Runs NODE_ENV=production node --import ./dist/sentry-init.js dist/index.js — loads Sentry hooks before starting the app. The server reads PORT from environment variables (Railway injects this automatically).
